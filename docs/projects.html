Surgery Duration Predictor

Machine Learning · Healthcare · University of Waterloo

A Random Forest regression model trained on ~35,000 historical surgical cases from a Canadian teaching hospital. Predicts OR surgery duration from procedure description text, surgical priority, patient type, room, and specialty — outperforming surgeon-reported booked durations on both RMSE and MAE.

Impact

  • Applied to 7,804 surgeries across 12 specialties (Mar 2024 – Mar 2025)
  • Reduced net OR scheduling cost by $363,898 (from $8.1M → $7.8M)
  • Created capacity for 176 additional high-volume procedures without extending OR hours

Stack

Python · scikit-learn · Random Forest · TF-IDF + SVD · Streamlit · Docker · FastAPI

Forecasting Rental Bike Count

Machine Learning · Time Series · Kedro Pipeline

A production-ready ML pipeline that predicts next-hour rental bike demand using CatBoost, lag features, and a live Plotly Dash dashboard. Built with Kedro for modular, reproducible workflows and containerised with Docker.

Highlights

  • Chronological 80/20 train-test split with no data leakage
  • Lag features up to 23 hours to capture same-time-yesterday demand patterns
  • Live inference simulation replaying historical data in real time
  • Supports CatBoost, Random Forest, and Linear Regression — switchable via config

Stack

Python · Kedro · CatBoost · scikit-learn · Plotly Dash · Docker · uv · ruff

Flight Delay Prediction

Machine Learning · Classification · Imbalanced Data

A binary classification model predicting severe carrier delays (60+ min) across 484,551 flights. Tackles heavy class imbalance (7.7% positive rate) using XGBoost with resampling strategies including RandomOverSampler, RandomUnderSampler, and ADASYN.

Highlights

  • 484K flights, 4,073 features after one-hot encoding
  • Demonstrates why accuracy is misleading for imbalanced problems (92.3% accuracy, near-zero F1 at baseline)
  • Experiments with three resampling strategies to improve minority class recall

Python · XGBoost · scikit-learn · imbalanced-learn · pandas
